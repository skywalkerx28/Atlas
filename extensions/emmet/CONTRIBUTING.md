## How to build and run from source?

Read the basics about extension authoring from [Extending Atlas](https://github.com/skywalkerx28/Atlas)

- Read [Build and Run Atlas from Source](https://github.com/skywalkerx28/Atlas/wiki/How-to-Contribute#build-and-run-from-source) to get a local dev set up running for Atlas
- Open the `extensions/emmet` folder in the Atlas repo in Atlas
- Press F5 to start debugging

## Running tests

Tests for Emmet extension are run as integration tests as part of Atlas.

- Read [Build and Run Atlas from Source](https://github.com/skywalkerx28/Atlas/wiki/How-to-Contribute#build-and-run-from-source) to get a local dev set up running for Atlas
- Run `./scripts/test-integration.sh` to run all the integrations tests that include the Emmet tests.
